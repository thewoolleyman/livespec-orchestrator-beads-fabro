# tests/dev-tooling/

Tests for the impl-beads-private enforcement scripts under
`<repo-root>/dev-tooling/`. Mirrors that tree one-to-one
(`check-tests-mirror-pairing`): each script `dev-tooling/checks/<name>.py`
has a paired `tests/dev-tooling/checks/test_<name>.py` covering both its
pass and fail cases.
